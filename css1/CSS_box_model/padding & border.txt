2. padding tidak bisa di kasih nilai auto

nilai border:
border: width style color;

note: border mempengaruhi ukuran kotak yang sudah di buat


style dari border:
1. solid 
2. dotted (titik-titik)
3. dashed (putus-putus)
4. double (dobel)
